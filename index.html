Projects

Here are some of the projects I've worked on:

Irish Mortgage Calculator

This is a simple mortgage calculator for Ireland. It allows users to calculate their monthly mortgage payments based on the loan amount, interest rate, and loan term.

It is built using React, TypeScript, and Fluent UI v8, and is hosted on Azure Static Web Apps.


Goals Bingo

This is a simple bingo game for New Year's Resolutions. It allows users to create a bingo card with their goals and share it with friends.
